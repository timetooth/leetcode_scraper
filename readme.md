## How to use

1. Use Leet problem set file to generate a problem set csv.
2. Problem set csv file is used to scrape detailed description, stats etc. for each problem.
3. Use Leet solutions file and problem set csv file to get top 100 solutions for every problem problem sorted by number of votes recieved.
